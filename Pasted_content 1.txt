Build a fully functional web platform called “Decoded Housing” that helps users find affordable housing, specifically:

- ARCH housing
- MFTE units
- Section 8–accepting apartments
- Transitional housing
- Shelters

The platform should function similarly to Apartments.com or Zillow, but focused entirely on income-restricted and hidden housing inventory.

---

## CORE GOAL

Create a searchable, filterable housing database that allows users to:

- Search by city (starting with Bellevue, WA)
- Filter by housing type (ARCH, MFTE, Section 8, Transitional, Shelter)
- View property-level listings (not just available units)
- Access contact information and application instructions
- See insider notes (e.g., “must ask for ARCH,” “waitlist only”)

---

## DATA INTEGRATION

Use Airtable as the backend database.

Import and structure the Bellevue ARCH/MFTE dataset from the provided spreadsheet.

Create the following Airtable tables:

### 1. Properties Table
Fields:
- Property Name
- City
- Address
- Housing Types (multi-select: ARCH, MFTE, Section 8, Transitional, Shelter)
- Unit Types (Studio, 1BR, 2BR, etc.)
- Year Built (if available)
- Total Units
- Affordable Units (if available)
- Contact Email
- Contact Phone
- Website URL
- Notes (free text)
- Verified Status (Yes/No)
- Last Contacted Date

---

### 2. Availability Intelligence Table
Fields:
- Property (linked)
- Has Waitlist (Yes/No)
- Waitlist Details
- Upcoming Units (Yes/No)
- Last Verified
- Source (user submission, direct call, etc.)

---

### 3. User Submissions Table
Fields:
- Property
- User Notes
- Date Submitted
- Contact Outcome (no response / waitlist / available / denied)
- Optional: User Email (for follow-up)

---

Design a clean, modern interface inspired by Zillow/Apartments.com:

### Homepage
- Search bar: “Enter city or ZIP”

---

### Search Results Page
- Grid + map layout
- Property cards showing:

  - Location
  - Housing types (badges)

  - City
  - Housing type

---

### Property Detail Page
Include:
- Full property info
- Housing types available
- Contact info

---

### Scripts Section
A dedicated page with exact scripts users can copy:

Example:

“Hi, I’m looking for an ARCH income-restricted unit. Do you currently have any available or a waitlist I can join?”

---

### “How It Works” Page
