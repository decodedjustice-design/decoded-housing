# DECODED HOUSING — PROJECT IDENTITY

## Purpose
Decoded Housing is a standalone platform focused on housing stability for King County residents.

It helps users:
- Find affordable housing
- Navigate housing applications
- Access basic needs (food, utilities, furniture, rent assistance)
- Understand tenant rights in plain language

---

## Strict Separation

This project is NOT part of Decoded Justice.

Do NOT include:
- Civil rights case management
- Legal evidence systems
- Police misconduct workflows
- Litigation tools or court-related systems
- Any reused components, naming, or logic from Decoded Justice

---

## Core Features Allowed

- Housing search and filtering
- Step-by-step application guidance
- Basic Needs Hub:
  - Food
  - Utilities
  - Furniture
  - Rent assistance
- Tenant rights education (simplified, non-litigation)
- Emergency shelter resources
